Manual de uso - Comandos Git e suas Funções

Este site tem como objetivo de ser um How to use de Git e GitHub. Use-o como um Guia rápido.

init Cria um repositorio Git vazio ou reutiliza um já existente.
clone Clona um repositorio em um novo diretorio.
branch Lista, cria ou remove Branches.
checkout Troca Branches ou restaura arquivos da Working Tree.
status Mostra o status da Working Tree.
diff Mostra a diferença entre commits, commit e Working Tree.
add Adiciona os conteudos do arquivo para o index.
commit Registra alterações no repositorio.
push Atualiza as referencias remotas junto com os objetos associados.
pull Busca e integra com outro repositorio ou uma branch local.
revert Reverte algumas commits existentes.
merge Adere dois ou mais historicos de desenvolvimentos juntos.
stash Guarda as alterações em um diretorio de trabalho.
rm Remove arquivos da Working Tree e do index.
config Pega e seta opções globais ou do repositorio.
reset Retorna a HEAD atual para o estado especificado.
remote Gerencia conjutos de repositorios localizados.
fetch Realiza o download de objetos e refs de outro repositorio.
show Mostra varios tipode de objetos.
help Mostra informação de ajuda sobre o Git.
rebase Reaplica commits em cima de outra base.
blame Mostra qual revisão e autor modificou cada linha de codigo pela ultima vez.
tag Cria, lista, apaga ou verifica a tag objeto assinada com GPG.

Este é um projeto pessoal e não tem o intuito de substituir a documentação original das tecnologias supracitadas. Para melhores informações, acesse o site Git e GitHub e saiba mais.
